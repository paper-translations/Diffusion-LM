# Diffusion-LM

Chinese translation of the paper *Diffusion-LM Improves Controllable Text Generation* (NeurIPS 2022).

## Contents

- `neurips.tex` — main LaTeX source
- `methods.tex`, `methods_tatsu.tex` — methods section drafts
- `anthology.bib` — bibliography
- `fig/` — figures

## Build

```bash
xelatex neurips.tex
xelatex neurips.tex
```
